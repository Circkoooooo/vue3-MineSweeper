# <center> MineSweeper

--- 
This is a minesweeper game, which build by `Vue@next`.


## How To Play
###  Online
It's deploy on the Netlify, so you can play it on the [Netlify website](https://hyx-minesweeper.netlify.app/).
### Locale
```
npm install

npm run dev
```

## UI Framework
* [Unocss](https://github.com/unocss/unocss)
